Liferay Hotels, is a demo that pretends to be an hotel portal. It includes a public portal and a personalization based on the customer rooms preferences along with a propor example liferay design systems.
You will find all of the required assets and fragments which we managed to build this portal with . Using this portal we managed to prove to our customers how easy we can build elegant portals using minimal effort.
This demo is targeting:
    CMS
    Personalization
    Design Systems
        Master Pages
        Style Books
        Fragments
